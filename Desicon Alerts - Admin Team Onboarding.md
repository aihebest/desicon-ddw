**Subject:** You've been set up to post on Desicon Alerts — here's how

Dear [Name],

You've been added to the **Desicon Alerts** admin team, which means you can publish announcements that appear directly on employees' screens across the company. This note explains how to get started.

**Your portal:** https://alerts.desiconapp.com

Open that link in Edge or Chrome and **sign in with your Desicon Microsoft 365 account** (your normal work email). Your access is already set up — only approved admins (HR, HSE, ICT, Management and selected Project Managers) can open this portal; everyone else is blocked.

**To send an announcement:**

1. Click **New Announcement**.
2. Enter a **Title** and **Message**.
3. Choose a **Category** (e.g. HSE, ICT, HR, General) and a **Priority**.
4. Choose **who should receive it** — Everyone, or a specific Location (Lagos, Port Harcourt, Abuja, Project Site), Department, or Project.
5. (Optional) Tick **Require acknowledgement** for important notices, so staff must confirm they've seen it.
6. Click **Publish.** It reaches matching staff automatically — no action needed from them.

**A note on priority — please use it thoughtfully:**

- **Normal / High** — appears as a pop-up at the bottom-right of the screen.
- **Critical** — takes over the full screen and must be acknowledged. Reserve this for genuine emergencies and urgent HSE notices only.

**To track engagement:** the **Dashboard** shows how many people each announcement was delivered to, read, and acknowledged. **History** lists everything previously sent.

Because these messages appear on screens company-wide, please keep them clear and professional, and target them to the right audience so people only get what's relevant to them.

If you have any trouble signing in or posting, contact the ICT Help Desk at **[helpdesk email / extension]**.

Welcome aboard,

## Quick reference card (optional — paste into the email or share separately)

| Field | What it does |
|-------|--------------|
| **Title / Message** | The headline and body shown in the pop-up. |
| **Category** | Labels the notice (HSE, ICT, HR, General…) and sets its colour. |
| **Priority — Normal/High** | Pop-up at bottom-right of the screen. |
| **Priority — Critical** | Full-screen, must be acknowledged. Emergencies/HSE only. |
| **Require acknowledgement** | Staff must click "Acknowledge"; tracked on the Dashboard. |
| **Audience** | Everyone, or by Location / Department / Project. |
| **Dashboard** | Delivered / Read / Acknowledged counts per announcement. |
| **History** | Everything published so far. |
